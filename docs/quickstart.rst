Development environment
***********************

You can use any python development environment to build and run the
models as well as analyse the results. The recommended tool is the
`Jupyter Notebook <http://jupyter.org/>`_. It allows you to do any of
this tasks in one document. Additionally you can clearly structure and
document any steps you have done.
